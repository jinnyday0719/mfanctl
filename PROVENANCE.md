# Provenance

mFanCtl does not include Macs Fan Control source code, bundled resources, icons,
translations, or extracted data files.

The SMC keys and model mappings in this repository are intended to be maintained
from local hardware probing, public macOS model identifiers, user-submitted
diagnostics, and runtime validation against available SMC keys.

Apple SMC interfaces are undocumented and may change across hardware and macOS
versions. Sensor profiles should be treated as best-effort compatibility data,
not as an Apple-supported API contract.
